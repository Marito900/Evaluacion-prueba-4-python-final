Este proyecto corresponde a la evaluacion 4 de Fundamentos de Programacion.

El programa permite administrar una biblioteca mediante un menú desarrollado en Python.

- Agregar libro.
- Buscar libro.
- Eliminar libro.
- Actualizar disponibilidad.
- Mostrar libros.
- Salir del sistema.

El programa valida que:

- El título no esté vacío.
- Las copias sean un número entero mayor o igual a 0.
- El período de préstamo sea un número entero mayor que 0.

- Python 3
